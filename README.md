# AI Browser Agent

Автономный AI-агент для автоматизации браузера на основе Claude AI.

## Быстрый старт

```bash
# 1. Очистка неиспользуемых файлов от старой версии
make cleanup

# 2. Установка зависимостей
make tidy

# 3. Настройка .env
cp .env.example .env
# Отредактируйте .env и добавьте AI_API_KEY

# 4. Запуск
make run
```
